# ✨ About

Lunaria merupakan bot Discord modern yang dikembangkan menggunakan **Discord.js v14** dan **MongoDB** dengan fokus pada performa, stabilitas, keamanan, serta kemudahan konfigurasi.

Bot ini dirancang agar satu bot saja mampu menggantikan berbagai bot populer seperti moderation bot, ticket bot, confession bot, logging bot, AI bot, economy bot, hingga RPG bot.
